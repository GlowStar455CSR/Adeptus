# 欢迎来到 仙舟通鉴

欢迎来到仙舟通鉴，这是一个由来自米哈游节奏交流站的纯良米友自发组织的属于纯良米友的 Wiki
## 这里有什么？
在这里你可以看到与米家节奏有关的内容，包括但不限于黑称、梗、魔怔人等
## 我该怎么快速找到自己想要看的内容？

如果你想看的条目已经出现在了导航栏里，可直接访问
