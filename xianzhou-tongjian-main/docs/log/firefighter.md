# 消防员事件
## 解释
一位退伍并且全身瘫痪的消防员加入了B站成为了一名up主，还考取了心理咨询证，并在B站上上传他用嘴玩游戏的视频来激励广大网友，可偏偏就有那么一些人吧，祂不识相，就在这个时候去冲击这位可怜的up主的视频，并且开盒了他的全部个人信息，天天打骚扰电话，使得激起了全网的公愤。
